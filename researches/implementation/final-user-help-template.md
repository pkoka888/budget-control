## Template Generation Rules

### Personalization Algorithm
```
1. Data-Driven Customization
   - Use actual financial data for calculations
   - Reference real transaction patterns
   - Include specific CZK amounts and dates
   - Adapt language to user proficiency level

2. Contextual Relevance
   - Prioritize user's stated goals and concerns
   - Include location-specific information (Prague vs regions)
   - Adapt to life stage (single, family, career level)
   - Consider risk tolerance and time horizon

3. Progressive Disclosure
   - Start with executive summary and key insights
   - Provide detailed analysis in logical sections
   - Include actionable next steps throughout
   - End with comprehensive action plan

4. Emotional Intelligence
   - Celebrate achievements and progress
   - Acknowledge challenges empathetically
   - Provide encouragement and motivation
   - Use positive, actionable language
```

### Content Generation Guidelines
```
1. Accuracy & Transparency
   - All calculations based on verified data
   - Clearly state assumptions and limitations
   - Provide confidence levels for projections
   - Include disclaimers for financial advice

2. Action-Oriented Focus
   - Every section includes specific, actionable steps
   - Prioritize high-impact, low-effort actions first
   - Provide timeframes and deadlines
   - Include accountability measures

3. Educational Approach
   - Explain financial concepts in IT terms
   - Provide reasoning for recommendations
   - Include learning resources and next steps
   - Encourage continuous financial education

4. Compliance & Ethics
   - No unauthorized financial advice claims
   - Clear data privacy and security messaging
   - Transparent about AI limitations
   - Encourage professional consultation when needed
```
